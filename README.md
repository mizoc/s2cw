# s2cw
convert string to morse code(QOD1)

## installtion
```bash
$git clone https://github.com/mizoc/s2cw && cd $_
$make
$sudo make install
```

## Usage
```bash
$s2cw hello
・・・・ ・ ・ー・・ ーーー
$echo 'CQ @7.001MHz'|s2cw
ー・ー・ ーー・ー   ・－－・－・  ーー・・・ ・ー・ー・ー ーーーーー ーーーーー ・ーーーー ーー ・・・・ ーー・・
$cw2s '...- ...- ...-  .... . .-.. .-.. ---'
VVV
```
